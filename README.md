## Featured Projects

### ML-Based Email Spam Detector

A machine learning application for email spam and ham classification using TF-IDF, Random Forest, and Logistic Regression CV. The system also includes OCR-based image spam detection for hidden text in image attachments.

- Live Demo: https://email-spam-detector-1003.streamlit.app
- Repository: https://github.com/Ghulam-Muhayyu-Din/Email-Spam-Detector
- Tech Stack: Python, Streamlit, Scikit-learn, Tesseract OCR, TF-IDF
- Research-based implementation connected with published work.

**Tech Stack:** Python, Streamlit, Scikit-learn, TF-IDF, Random Forest, Logistic Regression, OCR

---

### AI-Driven Digital Twin Framework for Cultural Symbols

A TensorFlow-supported AI labeling and detection workflow for cultural symbol extraction, classification, and transformation into a Digital Twin Symbol Library for preservation and generative reuse.

**Tech Stack:** Python, TensorFlow, Digital Twin, Computer Vision, Data Labeling

---

### Digital Twins for Intelligent Connected Vehicles

A reference architecture for vehicle Digital Twins integrating edge real-time control and cloud predictive analytics for intelligent connected autonomous and conventional vehicles.

**Research Areas:** Digital Twin, autonomous vehicles, edge-cloud systems, predictive analytics, safety-grade simulation

---

### Advanced Optical Sensing for Battery Safety

A sensor-fusion and machine-learning-driven battery health estimation model using FBG-based temperature and strain monitoring for lithium-ion battery safety.

**Research Areas:** FBG sensors, battery safety, predictive maintenance, edge processing, cloud dashboard
